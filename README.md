Sistema de Estoque em C#

Projeto de console desenvolvido em C# com foco em prática de lógica de programação, orientação a objetos e manipulação de coleções.

Funcionalidades
- Adicionar produtos
- Remover produtos
- Resumo do estoque
- Cálculo de valor total
- Produto mais caro/barato
- Média de preços
- Tratamento de exceções

Tecnologias
- C#
- .NET
- LINQ

Conceitos praticados
- Programação Orientada a Objetos
- Collections
- Exceptions
- LINQ
- Estruturas de repetição
- Estruturas condicionais
